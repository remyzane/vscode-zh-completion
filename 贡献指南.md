# 贡献指南

## 插件开发

### 创建插件

```bash
# 安装 Yeoman 工具集
npm install -g yo

# 安装 Code 扩展生成器
npm install -g generator-code

# 创建（本项目）
yo code
```

### 调试插件

安装 esbuild 插件: esbuild Problem Matchers（connor4312.esbuild-problem-matchers）

调试： VSCode 命令栏 -> 开始调试 -> Run Extension

### 生成插件

```bash
# 安装打包环境
npm install -g vsce

# 生成插件
vsce package --no-dependencies

# 发布插件
./publish.sh
```
