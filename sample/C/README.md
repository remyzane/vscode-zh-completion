# C语言中文编程样例

本目录包含了与 Python 版本对应的 C 语言中文编程样例。

## 文件结构

```
sample/C/
├── main.c              # 主程序文件
├── 中文包/
│   ├── 中文模块.h       # 头文件
│   └── 中文模块.c       # 模块实现
├── Makefile           # 编译脚本
└── README.md          # 说明文档
```

## 功能说明

### 主要特性

1. **中文变量名和函数名**：所有标识符都使用中文
2. **结构体替代命名元组**：用 `学生元组` 结构体替代 Python 的 namedtuple
3. **枚举类型**：用 `颜色枚举` 替代 Python 的 Enum
4. **函数指针实现多态**：用函数指针实现类似 Python 类继承的行为
5. **错误码处理异常**：用返回错误码的方式替代 Python 的异常处理

### 核心函数

- `中文函数()`: 演示枚举、结构体和参数处理
- `遍历列表()`: 演示循环和数组操作
- `匹配示例()`: 演示 switch-case 语句
- `计算面积()`: 演示函数参数和条件判断
- `动物类_初始化()`: 演示面向对象风格的实现
- `犬类_初始化()`: 演示继承的实现
- `除法运算()`: 演示错误处理
- `生成平方数列表()`: 演示动态内存分配

## 编译和运行

### 编译

```bash
make
```

### 运行

```bash
make run
```

或者：

```bash
./main
```

### 清理

```bash
make clean
```

## 编译要求

- GCC 编译器
- 支持 C99 标准

## 与 Python 版本的对比

| Python 功能 | C 语言实现 |
|------------|-----------|
| namedtuple | struct 结构体 |
| Enum 枚举 | enum 或 typedef enum |
| 类继承 | 函数指针 + 结构体嵌套 |
| 异常处理 | 返回错误码 |
| 列表推导式 | for 循环 + 动态内存分配 |
| match 语句 | switch-case 语句 |
| 动态类型 | 显式类型声明 |

## 学习要点

1. **内存管理**：C 语言需要手动管理内存分配和释放
2. **类型安全**：C 语言是静态类型语言，编译时检查类型
3. **指针操作**：函数指针实现多态，结构体指针访问成员
4. **编译流程**：从源代码到可执行文件的完整编译过程
5. **中文标识符**：支持中文变量名和函数名，提高代码可读性
