# JavaScript 中文命名示例

## 环境准备

请先安装本插件和 [JavaScript 语言支持插件](https://marketplace.visualstudio.com/items?itemName=ms-vscode.vscode-typescript-next)

## 编写代码

- 安装完成后，就可以使用中文命名了（如：类名、方法名）；
- 本插件将为您提供：中文标识符的快速自动补全。

## 整体效果

整体表现优秀：

- **JavaScript 语言本身**对中文命名具有良好的支持；
- **JavaScript 语言插件**对补全接口的实现也很完善。

## 验证示例

本项目中的示例代码已通过接受器验证，可正常运行。

### 运行命令

```bash
node 样例.js
```
