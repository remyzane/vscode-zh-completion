# JSON 中文命名示例

## 环境准备

只需安装本插件即可（VSCode 内置 JSON 支持）

## 定义 Schema

- 根据 [JSON Schema](https://json-schema.fullstack.org.cn/docs) 规范文档，
编写中文的文档结构、约束和数据类型定义（如：中文属性名、中文枚举项）

## 编写 JSON

- 定义 Schema 后，就可以使用中文属性名、中文枚举项了。
- 本插件将为您提供：中文标识符的快速自动补全。

## 整体效果

整体表现优秀：

- **JSON 规范本身**对中文命名具有良好的支持；
- **VSCode 内置的 JSON 实现**对补全接口的实现也很完善。
