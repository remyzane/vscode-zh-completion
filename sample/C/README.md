# C 语言 中文命名示例

## 环境准备

请先安装本插件和 [C 语言支持插件](https://marketplace.visualstudio.com/items?itemName=llvm-vs-code-extensions.vscode-clangd)

_请勿使用 cpptools（补全功能不完整）_

## 编写代码

- 安装完成后，就可以使用中文命名了（如：类名、方法名）。
- 本插件将为您提供：中文标识符的快速自动补全。

## 验证示例

本项目中的示例代码已通过编译器验证，可正常编译与运行。

### 编译要求

- GCC 编译器
- 支持 C99 标准

### 编译命令

```bash
make
```

### 运行命令

```bash
make run
```

### 清理命令

```bash
make clean
```
